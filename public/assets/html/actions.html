ACTIONS

Unlock Map

Remove the lock on the current map. The next user to update the map or an artifact on the map, will lock the map again for the other users.

Save hidden tiles

Saves the current state of the map as “hidden tiles,” which will be preserved in when resetting the map.

Soft Reset the map

Reset all the map tiles except the hidden tiles

Hard Reset the map

Resets all map tiles and clear the hidden tiles
