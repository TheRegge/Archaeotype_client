ACTIONS

Map Access

Unlock Map

Remove the lock on the current map. The next user to update the map or an artifact on the map, will lock the map again for the other users.

Covering tiles

Save Cleared Tiles

Saves the current state of the map's cleared tiles, which will be preserved in when doing a soft reset the map.

Soft Reset the Map

Reset all the map tiles except the previously saved cleared tiles.

Hard Reset the Map

Resets all the tiles on this map.

Artifacts

Randomly Rotate Artifacts

Randomly rotate artifacts on the map which are not already rotated.
